####代理模式
* 概念
    - 真实对象，被代理的对象
    - 代理对象
    - 代理模式，代理对象代理真实对象，达到增强真实对象的目的
* 实现方式
    - 静态代理，有一个类文件描述代理模式
    - 动态代理，在内存中形成代理类
        - 实现步骤
            - 代理对象和真实对象实现相同的接口
            - 代理对象 = Proxy.newProxyInstance();
            - 使用代理对象调用方法
        - 增强方式
            - 增强参数列表
            - 增强返回值类型
            - 增强方法体执行逻辑
