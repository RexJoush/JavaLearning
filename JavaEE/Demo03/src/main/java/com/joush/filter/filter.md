## MVC开发模式
#### Model 模型，JavaBean
* 完成具体的业务操作，如查询数据库，封装对象
#### View 视图，Jsp
* 展示数据
#### Controller 控制器，Servlet
* 获取请求参数
* 调用模型
* 将数据交给视图展示

##Filter 过滤器
####概念
* Web中的过滤器
    - 当服务器访问资源时，过滤器可以将请求拦截下来，完成一些特殊的功能
* 过滤器的作用
    - 一般用于完成通过的操作，如：登录验证，统一编码处理，敏感字符过滤
####快速入门
* 步骤
    - 定义一个类，实现接口 Filter
    - 复写方法
    - 配置拦截路径
        - web.xml配置
        - 注解配置
